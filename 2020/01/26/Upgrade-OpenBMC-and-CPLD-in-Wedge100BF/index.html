Upgrade OpenBMC and CPLD in Wedge100BF

Prepare

  • Edgecore Wedge100BF-32X
  • USB with at least 4Gb
  • OpenBMC and CPLD upgrade files

Please unzip all files and copy to USB root directory

Note: If you need upgrade files, please contact switch vendor or following contact window

Edgecore Network: jimmy_ou@edge-core.com

*Warning: It will be sugget to contact switch vendor to upgrade firmware, if user upgrade fail then switch might be brick. The switch will need to RMA and the vendor might charge for this.*

Steps

connect to console

After OpenBMC is ready, loginin with root/0penBmc

Upgrade OpenBMC

Confirm original OpenBMC version

Check MTD configuration

Install new OpenBMC image(flash-mavericks) to MTD(flash0)

and Reboot

Upgrade CPLD

Confirm original CPLD version

Set CPLD_UPPER_JTAG_SEL and CPLD_JTAG_SEL

Install CPLD code

*Do not power off DUT during installation. System shall reboot automatically after updating lower CPLD code*
